# 司马光砸缸项目 - 使用指南

## 项目概述

本项目为"司马光砸缸"故事的AI视频制作脚本，包含完整的剧本、素材清单和分镜脚本，适用于Seedance 2.0视频生成平台。

**故事时长**：90秒（6集 × 15秒）
**风格定位**：中国传统水墨动画风格 + 现代动画色彩
**文化背景**：北宋时期，司马光幼年机智救人的典故

## 制作流程

### 第一步：素材生成

使用Nana Banana Pro或其他AI图像生成工具，按照素材清单中的提示词生成所需素材。

**优先级排序**：
1. **必须生成**（核心素材）：
   - C01-C04：司马光的四个状态
   - C05：伙伴孩童基本造型
   - S01-S09：所有场景素材
   - P01-P03：关键道具

2. **可选生成**（根据需要）：
   - C06-C09：其他孩童的表情状态
   - P04-P05：服饰细节展示

**生成注意事项**：
- 确保风格统一：所有素材使用相同风格前缀
- 保持角色一致：司马光的形象在各素材中要保持一致
- 道具连贯：水缸、石头在不同场景中外观一致

### 第二步：视频生成

使用Seedance 2.0平台按集生成视频。

**E01生成流程**：
1. 上传S01、C01、C05、S03到素材位置
2. 复制E01分镜中的Seedance Prompt
3. 设置时长15秒
4. 生成视频

**E02-E06生成流程**：
1. 上传@视频1（上一集视频）和对应图片素材
2. 复制对应集数的Seedance Prompt（注意：E02-E06都以"将@视频1延长15s"开头）
3. 设置时长15秒
4. 生成视频

**重要提示**：
- E02-E06必须使用"视频延长"功能才能保持连贯性
- 每集最多上传9张图片
- 确保素材编号与分镜脚本中的上传位置对应

### 第三步：视频剪辑（可选）

Seedance 2.0的视频编辑功能有限，大部分修改需要重新生成。如需剪辑：
1. 下载生成的6集视频
2. 使用外部剪辑软件（如剪映、Premiere）进行拼接
3. 添加片头、片尾、字幕等

## 常见问题

**Q: 生成的视频风格不统一怎么办？**
A: 确保所有素材使用相同的风格前缀，重新生成不一致的素材。

**Q: 两集之间的过渡不流畅？**
A: 检查是否正确使用了@视频1延长功能，尾帧描述要与下一集的开场衔接。

**Q: 角色外观在不同集数中不一致？**
A: 这是AI生成的常见问题，可以通过在每集Prompt中加入角色特征描述来改善。

**Q: 某集生成失败？**
A: 检查是否有敏感词汇（如"江湖人士"），尝试修改表达方式。素材数量是否超过限制（最多9张图片）。

## 技术参数参考

**风格关键词**：
```
Chinese traditional ink wash painting animation style mixed with vibrant colors and smooth character animation, similar to "Havoc in Heaven" aesthetic, Song Dynasty period setting
```

**镜头运动词汇**：
- 推镜头（Zoom in）
- 拉镜头（Zoom out）
- 摇镜头（Pan）
- 移镜头（Truck）
- 跟镜头（Follow）
- 环绕镜头（Orbit）
- 希区柯克变焦（Hitchcock zoom）
- 手持晃动（Handheld shake）
- 慢镜头（Slow motion）

**音乐风格建议**：
- 开场：笛子+古筝，轻快欢快
- 危机：鼓点+弦乐，紧张急促
- 思考：缓慢坚定旋律
- 高潮：激昂管弦乐
- 结局：温暖感人旋律
