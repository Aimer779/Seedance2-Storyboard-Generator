# 故事转视频脚本 - 通用转换工具

## 使用指南

本工具帮助您将任何故事（小说、短篇、真实事件）快速转换为结构化的视频脚本。

---

## 第二步：三幕式结构构建

### 第一幕：铺垫（约占20-25%）

**目标：** 建立世界观、介绍主角、展示现状

| 要素 | 内容 |
|------|------|
| **主角是谁** | 姓名、年龄、身份、性格特征 |
| **现状如何** | 主角目前的生活状态 |
| **渴望什么** | 主角的内在需求或外在目标 |
| **触发事件** | 什么事件打破了现状，推动故事开始 |

**第一幕结束标志：** 主角踏上旅程/做出决定/接受挑战

---

### 第二幕：冲突发展（约占50-60%）

**目标：** 展现主角的努力、挫折和成长

| 要素 | 内容 |
|------|------|
| **试炼之路** | 主角面临的2-3个主要挑战 |
| **中点转折** | 故事中间的重大转折点（胜利或失败） |
| **低谷时刻** | 主角看起来最不可能成功的时刻 |
| **最后冲刺** | 主角如何准备迎接最终挑战 |

**第二幕结束标志：** 进入最终对决/最终考验

---

### 第三幕：结局（约占15-20%）

**目标：** 展现结果、传达主题

| 要素 | 内容 |
|------|------|
| **高潮场景** | 最紧张/最关键的时刻 |
| **结局揭示** | 最终结果是什么 |
| **主题升华** | 观众应该从中领悟到什么 |
| **余韵** | 给观众留下什么情感或思考 |

## 故事类型特殊处理

### 悲剧/正剧
- 强调命运的必然性
- 结局留有余韵
- 情感层次递进

### 喜剧
- 节奏明快
- 误会和反转
- 夸张的表演/镜头

### 悬疑/惊悚
- 信息逐步释放
- 视觉暗示和伏笔
- 节奏递进紧张

### 爱情故事
- 建立情感连接
- 展示关系变化
- 高潮情感爆发

---

## 常见转换技巧

### 小说→剧本
- 删减内心独白（改为视觉呈现）
- 合并次要角色
- 压缩时间线
- 强化动作和对话

### 真实事件→剧本
- 确定核心戏剧线
- 创造代表性场景
- 塑造清晰的人物弧光
- 平衡真实与戏剧性

### 短视频改编
- 前5秒抓住注意力
- 每个场景有明确目的
- 强化视觉冲击
- 结尾有记忆点
